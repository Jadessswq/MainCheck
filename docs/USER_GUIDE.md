# Инструкция для сотрудника (работника)

> Этот файл предназначен для пересылки сотрудникам вместе с файлом `AI_Suggester.oxt`.
> Администраторы — см. `ADMIN_GUIDE.md`.

---

## Установка (один раз, 2 минуты)

1. Сохраните присланный файл `AI_Suggester.oxt` куда угодно (например, на Рабочий стол).
2. Откройте **LibreOffice Writer**.
3. В меню: **Сервис → Управление расширениями…** (или `Ctrl+Alt+E`).
4. Нажмите **Добавить**, выберите `AI_Suggester.oxt`, подтвердите.
5. Закройте и снова откройте LibreOffice.

Готово. На панели инструментов появится одна новая кнопка — **«AI: Улучшить текст»**.

---

## Как пользоваться

1. Открыть любой документ Writer.
2. Выделить фрагмент текста, который нужно проверить (абзац, страницу, весь документ — что угодно).
3. Нажать **«AI: Улучшить текст»** на панели инструментов.
4. Подождать несколько секунд.
5. Появится диалог со списком правок. Нажать **OK** — исправления будут внесены в документ
   как **отслеживаемые изменения** (зачёркнутый старый текст + новый, цветом).
6. Пройтись по правкам: **Правка → Отслеживать изменения → Управление**. Принять
   или отклонить каждую отдельно (или все сразу).

---

## Если что-то не так

- **Диалог не появляется или долго думает.** Модель отвечает 10–40 секунд — это нормально. Если больше минуты — обратитесь к администратору.
- **Сообщение «Все серверы недоступны».** Сервис временно недоступен, обратитесь к администратору.
- **Сообщение «Выделите фрагмент».** Нужно сначала выделить текст, потом нажимать кнопку.
- **После принятия правки появились цветные пометки, которых не было.** Это и есть Track Changes — так и задумано. Управление: **Правка → Отслеживать изменения → Управление → Принять все** (или выборочно).

При любых других вопросах — администратору. Больше никаких действий с расширением от вас не требуется.
